# Grundschul-Erweiterung entwickeln

## Kernprinzip

**Fundamentale Transformation – nicht Vereinfachung.**

Die Grundschul-Erweiterung ist keine gekürzte Sek-I-Version.
Sie wechselt den Zugangsmodus vollständig:

| Aspekt | Sek I / Sek II | Grundschule (Kl. 3–4) |
|--------|---------------|----------------------|
| Einstieg | Chat-Dialog, AB | Bild/Zeichnung, Sitzkreis |
| Leitfrage | abstrakt-analytisch | konkret-emotional |
| Erarbeitung | schriftliche Fallanalyse | Bildkarten, mündlich |
| Modelle | explizit eingeführt | kindnahe Begriffe, keine Kategorisierung |
| Sozialform | Gruppenarbeit + Präsentation | Sitzkreis + Gespräch |
| Schreibanteil | hoch | minimal bis keiner |

---

## Schritt 1 – Phasenwahl begründen

Vor dem Schreiben: Welche 2 Phasen der Ablaufstruktur eignen sich?

**Entscheidungskriterien:**
- Wo steckt der emotionale Kern des Themas? → guter Einstiegspunkt
- Wo wird das Hauptkonzept vermittelt? → guter Vertiefungspunkt
- Wo ist der Inhalt so konkret/erlebbar, dass Grundschulkinder direkt andocken können?

**Nicht geeignet:**
- Phasen mit hohem Abstraktionsgrad ohne Bildzugang
- Reine Reflexions- oder Transferphasen (zu individuell)

Begründung der Phasenwahl kurz dokumentieren (1–2 Sätze je Phase).

## Schritt 2 – Inhalt je Phase entwickeln

Für jede der 2 gewählten Phasen:

```

**Material:** [Bildkarte / Zeichnung / Gegenstand – beschreiben]

**Ablauf:**
1. [Sitzkreis / Betrachtung des Bildes]

4. [Offene Weiterführung]

**Methodische Hinweise:**
- [Hinweis 1 für Lehrkraft]
- [Hinweis 2 für Lehrkraft]

**Zeitumfang:** ca. [X] Min. (identisch mit Sek I/II)
```

## Schritt 3 – Einbettung prüfen

Checkliste vor Abgabe:
- [ ] Exakt 2 Phasen mit Grundschul-Erweiterung
- [ ] Phasenwahl begründet
- [ ] Abschnitte direkt nach Sek I/II-Differenzierung eingebettet (nicht als Anhang)
- [ ] Icon „Erweiterung Grundschule" markiert
- [ ] Kein abstrakter Fachbegriff ohne kindnahe Erklärung
