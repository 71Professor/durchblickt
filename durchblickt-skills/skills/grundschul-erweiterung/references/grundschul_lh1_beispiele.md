# Grundschul-Erweiterung LH1 – Beispiele

## Beispiel 1: Grundschul-Erweiterung in der Hinführungsphase

| [((Icon „Grundschule"))]{.mark}                                       |
|                                                                       |
| **Anpassung des Einstiegs:** Statt des Chat-Dialog-Beispiels          |
| (Arbeitsblatt 1) wird ein gezeichnetes Gespräch zwischen einem Kind   |
| und einem bekannten Sprachassistenten oder Lernroboter eingesetzt     |
| (z. B. Alexa, Siri, ein Lerntablet-Assistent). Diese Geräte sind      |
| vielen Grundschulkindern bereits aus dem Alltag vertraut und bieten   |
| einen unmittelbaren, konkreten Anknüpfungspunkt.                      |
|                                                                       |
| **Impulsfrage:** „Kann ein Computer dein Freund sein?\"               |
|                                                                       |
| Diese Frage funktioniert für 8--10-Jährige unmittelbar und emotional  |
| -- sie lädt zur spontanen Meinungsäußerung ein, ohne abstraktes       |
| Vorwissen vorauszusetzen.                                             |
|                                                                       |
| **Methodische Hinweise:**                                             |
|                                                                       |
| -   Das Bild/die Zeichnung wird im Stuhlkreis oder an der Tafel       |
|     gemeinsam betrachtet                                              |
|                                                                       |
| -   Die Kinder äußern sich spontan, die Lehrkraft notiert Stichworte  |
|     (Tafel/Flipchart): *„Was sagt das Kind? Was antwortet der         |
|     Computer? Wie fühlt sich das Kind?\"*                             |
|                                                                       |
| -   Offene Weiterführung: *„Was kann ein Computer? Was kann er        |
|     nicht?\"* -- Antworten werden gesammelt, ohne Bewertung           |
|                                                                       |
| -   Auf eine anonyme Kartenabfrage wird in dieser Altersstufe         |
|     verzichtet; stattdessen mündliche Runde im Sitzkreis              |
|                                                                       |
| **Zeitumfang:** ca. 10 Min. (identisch mit Sek I/II)                  |
|                                                                       |
| **Materialhinweis:** Ein einfaches, klares Schwarz-Weiß-Bild oder --  |
| wer kreativ genug ist - eine gezeichnete Szene (A4, projizierbar)     |
| genügt. Das Bild zeigt z. B. ein Kind, das zu einem Lautsprecher      |
| spricht, mit Sprechblasen für Frage und Antwort. Alternativ: Bild im  |

| **Erwartbare Antworten der Lernenden:**                               |
|                                                                       |
| **Zur Frage „Ist das eine Freundschaft?\"**                           |
|                                                                       |
| **Sek I:**                                                            |
|                                                                       |
| \- „Nein, weil die KI nicht echt fühlt."                              |
|                                                                       |
| \- „Irgendwie schon, sie ist immer da und hört zu."                   |
|                                                                       |
| \- „Nein, weil sie nur so tut als ob."                                |
|                                                                       |

|                                                                       |

## Beispiel 2: Grundschul-Erweiterung in der Erarbeitungsphase 2

| [((Icon „Grundschule"))]{.mark}                                       |
|                                                                       |
| **Anpassung der Erarbeitungsphase:** Statt der schriftlichen          |
| Szenario-Analyse mit Arbeitsblättern (AB 4a--d) wird mit einem        |
| Bildkarten-Set gearbeitet: Vier einfach gezeichnete Situationen       |
| zeigen Kinder in alltäglichen Momenten, in denen ein Bedürfnis        |
| sichtbar wird (z. B. ein trauriges Kind allein auf dem Pausenhof, ein |
| Kind das bei einer Aufgabe nicht weiterkommt, ein Kind das abends     |
| allein im Zimmer sitzt, ein Kind das eine Entscheidung treffen muss). |
|                                                                       |
| **Kindgerechte Leitfragen:** „Was braucht das Kind in diesem Bild     |
| wirklich?\" „Kann ein Computer das geben?\"                           |
|                                                                       |
| **Methodische Umsetzung:** Die vier Bildkarten werden nacheinander im |
| Sitzkreis gemeinsam betrachtet oder in Think-Pair-Share bearbeitet:   |
|                                                                       |
| -   *Think:* Jedes Kind schaut kurz allein auf das Bild und überlegt  |
|                                                                       |
| -   *Pair:* Kurzer Austausch zu zweit (1 -- 2 Min.)                   |
|                                                                       |
| -   *Share:* Meldungen im Plenum, Lehrkraft sammelt Antworten         |
|     mündlich                                                          |
|                                                                       |
| Die Lehrkraft führt dabei kein formales Bedürfnis-Modell ein, sondern |
| arbeitet mit kindnahen Begriffen wie: *braucht jemanden zum Reden /   |
| braucht Hilfe / fühlt sich allein / möchte etwas lernen*. Der         |
| Übergang vom konkreten Bild zur eigenen Erfahrung wird mit einer      |
| einfachen Frage ermöglicht: *„Kennst du das auch, dass du das         |
| brauchst? An wen wendest du dich dann?\"*                             |
|                                                                       |
| **Zeitumfang:** ca. 20 -- 25 Min.                                     |
|                                                                       |
| **Materialhinweis:** Vier Bildkarten (A5 oder A4, projizierbar oder   |
| ausgedruckt), klare Zeichnungen ohne Text, sodass Kinder die          |
| Situation selbst interpretieren können. Die Karten können dieselben   |
| vier Grundbedürfnisse wie AB 4a--d aufgreifen (Verbundenheit,         |
| Kompetenz, Gehört-werden, Orientierung), ohne die Kategorien explizit |
| zu benennen. Bilder im [Oder GS Arbeitsblatt?]{.comment-start id="23" |
